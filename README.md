# ![][logo] GOKZ Achievements
###### Discord Bot
## Commands

### gokz/help
Shows a list of all commands
___
### gokz/ping
Shows the response latency
___
### gokz/register
Sends a private message to link your Steam account to Discord
___
### gokz/bans `[limit]`
Shows last bans from **Global API**
___
### gokz/wr `<game-mode>` `<map-name>` `[run-type]`
Shows the world record of the selected map
___
### gokz/bwr `<game-mode>` `<map-name>` `[run-type]` `[b-number]`
Shows the bonus world record of the selected map
___
### gokz/maptop `<game-mode>` `<map-name>` `[run-type]` `[limit]`
Shows the global leaderboard of the selected map
___
### gokz/bmaptop `<game-mode>` `<map-name>` `[run-type]` `[b-number]` `[limit]`
Shows the global bonus leaderboard of the selected map
___
### gokz/jumptop `<jump-type>` `[bind/no-bind]` `[limit]`
Shows the global leaderboard of the selected jump type

## Command Parameters
### < _%param-name%_ >
Required parameter
___
### \[ _%param-name%_ \] 
Optional parameter
___
### Game mode
+ **KZT**
  - KZTimer
+ **SKZ**
  - SimpleKZ
+ **VNL**
  - Vanilla
___
### Map name
Any string of full map name
___
### Run type
+ PRO
+ TP
___
### B number
Bonus number
___
### Limit
Limit on the number of results given by the command
___
### Bind / No-bind
Was CJ bind used?
